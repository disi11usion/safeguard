## Project Description and Scope

This project focuses on building an AI-powered portfolio risk and scenario intelligence system within the Safeguard platform.

Students will help develop a portfolio module where users can add multiple assets (crypto, stocks, ETFs, gold) and view their overall risk exposure in a single screen. Instead of focusing on prices, the system emphasizes risk visibility and portfolio health.

Each asset will be analyzed using Safeguard’s existing modules, including:
- Social sentiment
- News impact
- Government/macro data
- User psychology

These outputs will be combined to generate:
- Simple risk levels (Low / Medium / High)
- A unified Portfolio Health score (Healthy / Stressed / Fragile)

The project also includes basic scenario-based stress testing, allowing users to simulate events such as:
- Market drops
- Panic situations
- Breaking news

This helps users understand how their portfolio might behave under stress.

An AI explanation layer will generate short, human-readable portfolio summaries, helping users understand key risks and exposures without providing buy/sell advice.

### Scope of Student Work

- Portfolio creation interface and basic frontend
- Integration of existing Safeguard analysis modules
- Portfolio Health calculation and visualization
- Simple stress-test scenarios
- AI-generated portfolio commentary and explanations

The project does **not** involve building new market data pipelines or prediction models. It focuses on integrating existing systems into a clear, user-friendly portfolio risk experience.
